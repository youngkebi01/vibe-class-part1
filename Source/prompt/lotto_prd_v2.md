# PRD: 로또 번호 추천 시스템

**v1.0 | 2026-05-28 | 상태: Draft**

---

## 1. 목적

1~45 중 6개를 뽑는 로또(6/45)에서 사용자가 제외/고정 번호를 설정하고, 자동·반자동·수동으로 번호를 추천받는 순수 바닐라(HTML/CSS/JS) 단일 파일 앱.

---

## 3. 번호 상태 & 볼 색상

**상태 순환**: `기본(흰색)` → `고정(강조색)` → `제외(흐림/취소선)` → `기본`

**볼 색상 (공식 기준)**:

| 범위 | 색상 |
|------|------|
| 1–9 | 노랑 `#FBC400` |
| 10–19 | 파랑 `#069CE7` |
| 20–29 | 주황 `#FF7272` |
| 30–39 | 회색 `#AAAAAA` |
| 40–45 | 초록 `#B0D840` |

## 5. 핵심 로직

```javascript
// 상태
const state = {
  mode: 'auto',          // 'auto' | 'semi' | 'manual'
  excluded: new Set(),
  pinned: new Set(),
  selected: new Set(),   // 수동 모드용
  history: []            // 최대 5회
};

// 번호 생성
function generate(pinned, excluded) {
  const pool = Array.from({length: 45}, (_, i) => i + 1)
    .filter(n => !excluded.has(n) && !pinned.has(n));
  const need = 6 - pinned.size;
  if (pool.length < need) throw Error('가능한 번호 부족');
  const picked = shuffle(pool).slice(0, need);
  return [...pinned, ...picked].sort((a, b) => a - b);
}

// Fisher-Yates 셔플
function shuffle(arr) {
  for (let i = arr.length - 1; i > 0; i--) {
    const j = Math.floor(Math.random() * (i + 1));
    [arr[i], arr[j]] = [arr[j], arr[i]];
  }
  return arr;
}
```

## 6. 예외 처리

| 상황 | 처리 |
|------|------|
| 제외 번호 ≥ 40개 | "최소 6개 번호 필요" 토스트 알림 |
| 고정 + 제외 번호 충돌 | 나중 선택이 덮어씀 + 알림 |
| 수동 6개 미만 생성 클릭 | "6개를 선택해주세요" 알림 |
| 가능한 풀 부족 | "제외 번호를 줄여주세요" 알림 |
