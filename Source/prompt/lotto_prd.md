## 1. 개요 (Overview)

### 1.1 배경
로또(Lotto 6/45) 구매자들은 매주 번호를 선택하는 과정에서 완전한 랜덤, 직접 선택, 혹은 통계 기반 선택 등 다양한 방식을 원한다. 본 시스템은 사용자가 원하는 조건(제외 번호, 고정 번호)을 설정하고 자동/반자동으로 번호를 추천받을 수 있도록 한다.

### 1.2 목적
- 순수 바닐라 JavaScript(HTML/CSS/JS)로 구현하는 단일 페이지 로또 번호 추천 도구 제공
- 외부 라이브러리 없이 동작하는 경량 클라이언트 앱 구현
- 수동, 반자동, 자동 세 가지 모드를 통해 다양한 사용자 니즈 충족

### 1.3 범위
- 프론트엔드 단독 구현 (서버 없음, API 없음)
- 단일 HTML 파일로 배포 가능
- 실제 당첨 예측 기능 없음 (확률적 랜덤 생성)

## 3. 기능 요구사항 (Functional Requirements)

| ID | 요구사항 | 우선순위 |
|----|----------|----------|

## 8. 예외 처리 (Edge Cases)

| 상황 | 처리 방식 |
|------|-----------|
| 제외 번호 > 39개 | 경고 토스트: "최소 6개 번호는 남겨야 합니다" |
| 고정 번호 = 6개 | 자동 생성 없이 고정 번호 6개를 결과로 바로 출력 |
| 고정 + 제외 번호 충돌 | 나중에 설정한 상태로 덮어씀 + 사용자 알림 |
| 수동 모드에서 6개 미만 선택 후 생성 클릭 | "6개를 선택해주세요" 경고 |

## 9. 구현 고려사항 (Implementation Notes)

### JavaScript 핵심 데이터 구조
```javascript
const state = {
  mode: 'auto',         // 'auto' | 'semi' | 'manual'
  excluded: new Set(),  // 제외 번호
  pinned: new Set(),    // 고정 번호
  selected: new Set(),  // 수동 모드 선택 번호
  result: [],           // 현재 결과 [n1, n2, n3, n4, n5, n6]
  history: []           // 이전 결과 이력 (최대 5)
};
```

### 번호 생성 알고리즘 (의사코드)
```
function generateNumbers(pinned, excluded):
  pool = [1..45] - excluded - pinned
  need = 6 - pinned.size
  if pool.size < need → throw Error
  random = shuffle(pool).slice(0, need)
  result = sort([...pinned, ...random])
  return result
```

### Fisher-Yates 셔플 사용
```javascript
function shuffle(arr) {
  for (let i = arr.length - 1; i > 0; i--) {
    const j = Math.floor(Math.random() * (i + 1));
    [arr[i], arr[j]] = [arr[j], arr[i]];
  }
  return arr;
}
```
